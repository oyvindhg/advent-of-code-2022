# advent-of-code-2022
Rust solutions to the [2022 Advent of Code](https://adventofcode.com/2022) problems.

### How to run

`cargo run -p day{DD}` where `{DD}` is the day number.

For example, `cargo run -p day01` runs the solution for the first day.
